This portion of the Behrend archives is a set of letters from Paul Siple to Mr. Behrend informing him whilst he is journeying to Antarctica. More specifically the DIGIT program has gone forward with this as a standalone project gather photos of the transcripted letters and archiving them digitally in a website for ease of access.
